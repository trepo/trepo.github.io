Property Graph

At the core of Trepo is a property graph. Any directed property graph may be used, provided it is supported by a vGraph implementation.

Java

The property graph implementation for java is Tinkerpop's Blueprints. It supports a large number of backends, including Neo4J, Titan, and many others.

Javascript

Because there was no real property graph implementation for Javascript, we're building jGraph. It works in Node.js and modern browsers.
