Trepo?

People need genealogy software that helps them work together. They need software that assists them as they work. They need software that works on their behalf. We as developers need the tools and resources to make that happen. Trepo is one of those tools.

Trepo is an open source genealogy API. It comes with a versioned genealogical database, a distributed query framework, and a set of extensible research and data models. All of this is wrapped in a language agnostic specification and supplemented with HTML5 Web Components.

Developers can easily implement collaboration, deal with disagreements, and control access to a person’s data. And because they are using a solid, extensible foundation, they can can focus on creating new experiences instead of recreating basic functionality.

Property Graph

A Directed Property Graph that is ID Addressable

A Directed Graph is where each Edge in the graph has a direction, going from one node to another. A Directed Property Graph has a set of key/value pairs on each Node and Edge called properties.

Property Graph Introduction

vGraph

A Distributable, Versioned Graph Database

In vGraph, every Node and Edge is assigned an immutable UUID and Label, and every change is tracked in a Commits. Different instances of vGraph can be linked together using Boundary Nodes.

vGraph Specification

xSearch

An Indexing and Query Specification

Specifies what is indexed and available for searching, as well as defining what queries look like. Search for People, Sources, and more using a flexible set of search labels.

xSearch Specification

pTree

An Extensible Genealogical Data Model

Defines a schema for storing genealogical information in vGraph. Store persons, events, facts, and more in an extensible data model designed to facilitate versioning and disagreements.

pTree Specification

nPipes

A Cross Repository Graph Traversal Framework

nPipes is an expressive query language wrapped in a language agnostic specification. It allows developers to ignore Boundary Nodes and supports several different execution strategies.

nPipes Specification

REST API

A REST API Specification

A language agnostic API specification letting the developer access vGraph, pTree, and yProof as well as providing several nice nPIpes traversals to get ancestry, descendancy, and more.
