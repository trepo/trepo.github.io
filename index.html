Trepo?

Trepo is a distributed, version controlled, genealogical database built atop a standard property graph. It is cross platform, language agnostic, extensible, scalable, and (best of all) completely open source under the MIT License!

Why?

  • Data should be versioned.
  • Control should be given to the user.
  • Access should be up to the user.
  • Clobbering should never happen.
  • Collaboration should be easy.
  • Sharing should be strait forward.
